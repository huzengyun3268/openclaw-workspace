## 🚀 上传发布步骤

### 第一步：下载开发者工具
👉 https://developer.douyin.com/ → 下载「字节跳动开发者工具」

### 第二步：导入项目
1. 打开开发者工具
2. 点击「导入项目」
3. 选择文件夹：`C:\Users\Administrator\.openclaw\workspace\game2048_douyin`
4. AppID 会自动填写：`tt646212dffa0348b102`
5. 点击确定

### 第三步：预览测试
1. 点击左侧「预览」
2. 用抖音扫码查看效果
3. 确认游戏可以正常运行

### 第四步：上传代码
1. 点击「上传」
2. 填写版本号：`1.0.0`
3. 填写备注：首发版本
4. 点击确认

### 第五步：提交审核
1. 打开 https://developer.douyin.com/
2. 进入「版本管理」
3. 找到刚上传的版本
4. 点击「提交审核」
5. 等待1-3个工作日
