# GitHub Setup Script
# Run this after creating a GitHub repository

# Step 1: Open terminal and login to GitHub
# C:\gh\bin\gh.exe auth login

# Step 2: After login, replace YOUR-USERNAME with your GitHub username
# Then run these commands:

# cd C:\Users\Administrator\.openclaw\workspace
# git remote add origin https://github.com/YOUR-USERNAME/openclaw-workspace.git
# git remote -v
# git push -u origin master
