# 情况说明

**关于临海市华侨地块希尔顿酒店项目（暖通）招标投诉的答复**

反映日期：2026年3月18日
答复日期：2026年3月25日

---

针对反映人提出的关于本项目招标文件设置"机电工程施工总承包一级资质"存在倾向性问题，经调查核实，现答复如下：

**一、关于资质设置依据**

本项目设置"机电工程施工总承包一级"资质条件，系根据**项目实际建设规模和管理需要**，结合希尔顿酒店作为国际五星品牌的运营标准，经综合评估后确定。该资质条件与项目施工技术难度、管理复杂度相匹配，不存在为特定企业量身定做的情况。

**二、关于市场竞争性**

经核查，浙江省内具备机电工程施工总承包一级资质的企业数量充足，市场竞争充分，具备充分的可选择性和竞争性。该资质条件设置符合《招标投标法》及其实施条例的相关规定，不构成以不合理条件限制或排斥潜在投标人。

**三、关于处理结论**

综合上述调查情况，反映人所反映的"招标资格具有倾向性"问题**情况不属实**。

本项目招标过程依法依规进行，全程接受社会监督。如反映人对本答复有异议，可在收到本答复之日起**30日内**向临海市住房和城乡建设局依法提出复查申请。
