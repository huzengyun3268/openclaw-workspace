# 股票信息查询技能

## 支持的查询类型

1. **实时行情** - 当前价格、涨跌幅、成交量等
2. **财务数据** - 每股收益、净资产、ROE等
3. **分红送股** - 历史分红、送转股记录
4. **资金流向** - 大单资金进出
5. **基本面分析** - 市盈率、市净率等估值指标

## 数据来源

使用 `web_fetch` 工具从以下网站获取数据：

- **东方财富** - https://quote.eastmoney.com/sh600352.html（股票代码格式：sh600352、sz000001）
- **新浪财经** - https://finance.sina.com.cn/realstock/company/sh600352/nc.shtml
- **雪球** - https://xueqiu.com/S/SH600352

## 查询步骤

### 1. 确定股票代码

- 沪市：以600、601、603开头，如 600352
- 深市：以000、001、002、300开头，如 000001、300750
- 北交所：以8、4开头，如 830799

### 2. 获取数据

使用 web_fetch 抓取页面：

```bash
# 东方财富行情页
https://quote.eastmoney.com/sh600352.html

# 东方财富财务页
https://emweb.securities.eastmoney.com/pc_hsf10/pages/index.html?type=web&code=SH600352&color=b#/cwfx
```

### 3. 分析数据

提取关键指标：
- **现价** - 当前交易价格
- **涨跌** - 价格变动
- **涨跌幅** - 百分比变动
- **成交量** - 成交数量
- **成交额** - 成交金额
- **市盈率(TTM)** - PE值
- **市净率** - PB值
- **总市值** - 公司市值
- **每股净资产** - BPS
- **ROE** - 净资产收益率

## 输出格式

查询结果以清晰的中文呈现，包含：
1. 股票基本信息（代码、名称、板块）
2. 实时行情数据
3. 关键财务指标
4. 投资建议（仅供参考，不构成投资建议）

## 注意事项

- 股市有风险，投资需谨慎
- 建议仅供参考，不作为买卖依据
- 数据可能存在延迟，请以实际交易为准
- 如需最新数据，建议用户查看实时行情
